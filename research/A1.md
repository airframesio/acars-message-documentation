# Label: A1

## Description

When over ATS with MTI `CLX` it is used for `Oceananic Clearance`.

## Examples

```
/PIKCLYA.OC1/CLX 0832 220611 EGGX
CLRNCE 602
DLH436 CLRD TO KORD VIA
ERAKA
RANDOM ROUTE
60N020W 61N030W 61N040W
60N050W TOXIT
FM ERAKA/0959 MNTN F380
M085
END OF MESSAGEE823
```

## Analysis

PIKCLYA = ATC Center Facility issuing the Oceanic Clearence
0832 = HHMM UTC Time Stamp
220611 = YYMMDD UTC Date Stamp
EGGX = Issuing Oceanic FIR / OACC (Shanwick)
DLH436 = Flight Number or Callsign
KORD = Destination Airport
ERAKA = Oceanic Route Entry (Way)Point
TOXIT = Oceanic Route Exit (Way)Point
FM = Flight Management
0959 = HHMM UTC Time over Entry (Way)Point
MNTN = Maintain
F380 = Flight Level (380=38000ft)
M085 = Mach Number (0.85)

60N020W......60N050W = Oceanic Route (Way)Points
